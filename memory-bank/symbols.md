- 标准引用: `[↗️σ₁:R₁]` - 指向项目简介的需求 1
- 模式引用: `[↗️Ω₁]` - 指向研究模式

- Δ₁ = 破坏性操作 → 警告 ∧ 确认 ∧ 备份

- Δ₃ = 重新初始化 → 警告 ∧ 确认 ∧ 备份
- Δ₄ = 错误处理 → 报告 ∧ 建议恢复
